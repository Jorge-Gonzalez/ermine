# ADR-0001 — Spacing exploration established the method

source: `context/MONKY-STYLE-GRAMMAR-PROJECT-HISTORY.md` lines 51–107

The first phase began with naming wider spacing. It established perceptual density naming,
one-axis-per-class discipline, and the distinction between semantic scope and deployment scope.
It also proposed a proportion axis and runtime value channel that later phases retired.

Affected current rulings: R-SPACE-01, R-SPACE-02, R-DENSITY-01, R-DENSITY-02,
R-DENSITY-03, R-DENSITY-04, R-PROPORTION-01.
