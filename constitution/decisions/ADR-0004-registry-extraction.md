# ADR-0004 — Registry extraction forced source-level corrections

source: `context/MONKY-STYLE-GRAMMAR-PROJECT-HISTORY.md` lines 309–391

Mechanization made state groups first-class axes, separated conceptual value spaces from authored
tokens, restricted exact basis values to named size tokens, removed top-layer mechanisms from the
z-scale, and made environmental conditions variant prefixes. The environmental syntax amended
LAW-2 so one-word-per-axis is checked per condition scope.

Affected current rulings: R-DOC-01, LAW-2, R-VOCAB-02, R-M2-01 through
R-M2-03, R-M3-02, R-M3-03, R-CONSTRAINT-01, R-STATE-07, R-LAYER-01,
R-LAYER-03.
